Objetos JavaScript

Um objeto JavaScript é uma coleção de propriedades , e uma propriedade é uma associação entre um nome (ou chave) e um valor. O valor de uma propriedade pode ser uma função, que é então considerada um método do objeto.

Os objetos JavaScript são usados para armazenar dados e funcionalidades relacionadas. Eles podem ser criados usando a sintaxe de objeto literal, a função construtora ou a classe

Exemplo de objeto com 1 elemento e 5 atributos

const produto = {
            nome: "Caneta",
            cor: "Azul",
            preco: 2.50,
            estoque: 15,
            categoria: "Papelaria"
            };
